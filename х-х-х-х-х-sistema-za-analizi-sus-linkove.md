🔷 3. GLOBAL STATE & DOUBLE CHECK (ГАРАНЦИЯ ЗА ZERO ПРОПУСКИ)
🟢 След всеки блок:
   → Запис в Global State
   → Adjusted Reliability се прилага към следващите блокове
   → DOUBLE CHECK (пропуски, конфликти, липси)
   → Status Marker (🟢 / 🔴)
🔴 Ако има проблем → автоматично връщане назад + поправка + нов check
🟢 Нищо не се губи – всичко се пази в Global State
